# Atlas Tech - Landing Page

Bienvenido al repositorio oficial de la landing page de **Atlas Tech**, una agencia especializada en soluciones de Inteligencia Artificial para potenciar el crecimiento y la eficiencia empresarial.

## 🚀 Misión
Ayudamos a los negocios a escalar mediante la implementación de automatización inteligente, análisis de datos y soluciones de IA a medida.

## ✨ Características Técnicas
- **Framework**: React / TypeScript.
- **Estilos**: CSS personalizado con variables de marca (Azul Atlas Tech).
- **Idioma**: Totalmente adaptado al español.
- **Assets**: Logotipo y favicon personalizados.

## 📁 Estructura del Proyecto
- `atlasws/`: Directorio principal del código fuente de la aplicación React.
  - `src/components/`: Componentes reutilizables (Hero, About, Services, etc.).
  - `src/main-component/`: Componentes de página y enrutamiento.
  - `src/images/`: Activos visuales y logotipo.
  - `src/css/`: Estilos globales y específicos del tema.

## 🛠️ Instalación y Uso
Para ejecutar el proyecto localmente, asegúrate de tener Node.js instalado:

1. Entra al directorio del proyecto:
   ```bash
   cd atlasws
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
