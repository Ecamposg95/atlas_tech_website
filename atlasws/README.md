# Atlas Tech - App

Este es el directorio principal del código fuente de la aplicación **Atlas Tech**.

## 🚀 Misión del Proyecto
Desarrollar una presencia web profesional y moderna para Atlas Tech, destacando nuestras capacidades en Inteligencia Artificial y automatización.

## 🛠️ Scripts Disponibles

En este directorio, puedes ejecutar:

### `npm start`
Inicia la aplicación en modo de desarrollo.\
Abre [http://localhost:3000](http://localhost:3000) para verla en el navegador.

### `npm run build`
Construye la aplicación para producción en la carpeta `build`.\
Optimiza la construcción para el mejor rendimiento.

## 📁 Detalles de Adaptación
Este proyecto fue adaptado de un template inicial para cumplir con los requisitos de Atlas Tech:
- **Traducción**: Todo el contenido fue traducido a español para el mercado hispanohablante.
- **Branding**: Sincronizado con la identidad visual de Atlas Tech (Blue Theme).
- **Estructura**: Simplificada para mejorar la navegación y el mantenimiento.
